# LeetCode Problems Tracker

A clean, modern web application to track your LeetCode problem-solving progress across 27 different categories and 405 problems.

## Features

- **405 Problems** organized across 27 categories
- **Interactive Toggle Buttons** - Mark problems as done/not done with a simple click
- **Progress Tracking** - Circular progress indicator showing overall completion
- **Search & Filters** - Find problems by name, difficulty, or completion status
- **Collapsible Categories** - Keep your view clean by expanding only what you need
- **Bulk Actions** - Mark all problems in a category as done/clear all
- **Auto-Save** - Progress is automatically saved to your browser's localStorage

## Getting Started

### Prerequisites

- Node.js 18+ or Bun
- npm, yarn, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/VasuML07/Leetcode_Revision.git

# Navigate to the project directory
cd Leetcode_Revision

# Install dependencies
npm install
# or
bun install

# Run the development server
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
